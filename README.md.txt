# Mortgage Calculator (Java)

A simple Java console application that calculates monthly mortgage payments based on user input.

## Features
- Takes principal amount
- Takes annual interest rate
- Takes loan period in years
- Displays mortgage in currency format

## Concepts Used
- Variables & constants
- User input using Scanner
- Math.pow()
- Number formatting
- Java basics

## How to Run
1. Compile the program:
   javac MortgageCalculator.java
2. Run the program:
   java MortgageCalculator

Principal: 100000  
Annual Interest Rate: 3.92  
Period (Years): 30
